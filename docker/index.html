TAuth Docker Playground

Sign in with Google via the shared mpr-ui header wired to a running TAuth instance. The demo relies on the Docker Compose setup in this repository, which pairs the front-end file server with the TAuth backend container.

API Base URL

The front-end reads the authentication base URL and Google Client ID from data-* attributes on the <body>. Update those values directly in this file if you run against a different backend or OAuth client.

  2.

    4. 
          
  4. Open http://localhost:8000.
